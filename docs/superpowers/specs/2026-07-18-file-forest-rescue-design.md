# 文件森林救援队设计

## 目标

用一局约 8–10 分钟的六次救援，把路径、命名、移动/复制、扩展名、分类和回收站恢复连接成真正可操作的文件管理模型。

## 解锁

- 完成文件森林五课：`file-home`、`name-your-work`、`move-and-copy`、`file-types`、`learning-backpack`。
- 游戏中心第十二张卡，锚点 `file-forest-rescue`。

## 六次救援

1. **路径寻路**：按根目录—学习—科学的文件夹路径找到目标文件。
2. **清楚命名**：把模糊名称换成包含主题的名称，同时保留 `.png` 扩展名并避免重名。
3. **移动归位**：把唯一原件移动到正确文件夹，验证原位置不再保留。
4. **复制备份**：把作品复制到备份文件夹，验证两个位置各有一份。
5. **类型分拣**：依据 `.png`、`.txt`、`.wav` 等扩展名线索分类；陌生类型不随意打开。
6. **回收站恢复**：从虚拟回收站恢复误删文件，再用搜索与文件夹结构确认位置。

## 规则与边界

- 所有文件和文件夹均为固定虚构数据，无本机文件访问、上传或自由输入。
- 删除只发生在虚拟回收站，可恢复；错误操作不破坏状态。
- 正确操作记录文件数量、路径和扩展名证据；双击不会重复复制或推进。
- 六关轮换重玩，无倒计时、排行榜或惩罚。

## 质量

- 纯状态机测试路径、命名、移动、复制数量、分类、恢复、双击与通关。
- 按需加载；44px 控件；移动单栏；当前文件有文字和边框；减少动画与强制颜色模式。
- 完整测试、类型、lint、审计、Pages 和线上验证。
