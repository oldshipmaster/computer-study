# 创作工坊项目赛设计

## 目标

把创作工坊五门课程组合成一节 8–10 分钟的六项目接力。孩子通过可视卡片做设计决策，并在右侧作品集看到每个决定的理由。

## 六个项目

1. 像素图纸：先确定网格轮廓，再用有限色板保持可读性。
2. 文档编辑：用标题、正文和留白形成信息层级。
3. 幻灯片导演：按开头、发现、解决、结尾安排故事，并精简单页内容。
4. 素材版权员：只用自制或明确允许的虚构素材，并留下来源说明。
5. 数据表设计师：一列一种数据、单位进表头，再根据问题选择合计。
6. 综合发布台：核对内容、版权、数据与观众，发布一份虚拟班级作品。

## 规则与安全

- 仅使用代码内固定的虚构画面、文字、表格和素材卡，不上传文件、不输入姓名。
- 每步三选一；错误保留当前作品并给观察提示，正确后记录可迁移的设计理由。
- 六个项目完成后轮换重玩顺序，不计时、不排行、不保存选择或错误。
- 完成创作工坊五门课程后解锁。

## 可访问性

- 所有选择为按钮，状态由文字与边框共同表达。
- 新步骤和新项目把焦点移至标题；进度、反馈与作品集有明确标签。
- 44px 触控目标，窄屏单列，支持减少动画与强制颜色。
