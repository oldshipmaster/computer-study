# 游戏中心发现与复玩升级设计

## 目标

十二种玩法集中后，孩子不需要逐张阅读才能决定玩什么。游戏中心增加“推荐三局”、主题筛选和“只看已解锁”，帮助二年级孩子自己探索，同时保留清楚的课程解锁路径。

## 交互

- 顶部展示最多三张仅来自已解锁玩法的推荐卡；“换一组”以稳定轮换算法改变顺序，不随机追踪孩子。
- 四个儿童可读主题：综合挑战、编程与逻辑、电脑与网络、安全与文件；另有“全部玩法”。
- “只看已解锁”是可撤销筛选。筛选结果为空时提供一键恢复，不隐藏学习路径。
- 每张卡继续显示解锁进度、时长与下一门前置课程。
- 筛选状态只在当前页面内存，不写入本机进度，也不上传。

## 推荐规则

- 只推荐当前已解锁游戏，最多三种且不重复。
- 优先跨主题轮换，避免三张都是同类玩法。
- 相同输入和轮换值产生相同结果，便于测试；异常轮换值归零。
- 新孩子只有探险任务牌已解锁时，只推荐这一种，不伪装成三种。

## 可访问性与边界

- 筛选使用真实按钮和 `aria-pressed`；结果数量用状态区域播报。
- 44px 触控目标、键盘可操作、窄屏单列、高对比度可辨认。
- 不增加倒计时、连续签到、抽奖或排行榜。
