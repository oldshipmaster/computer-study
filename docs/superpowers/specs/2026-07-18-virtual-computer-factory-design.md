# 虚拟电脑装配厂设计

## 目标

把硬件实验岛五课连接成一场约 8–10 分钟的虚拟装配任务。孩子通过信息流、部件职责、比特表示和安全排错建立完整的“电脑怎样工作”模型，而不是把零件名称孤立记忆。

## 解锁与安全边界

- 完成 `input-process-output`、`cpu-memory-storage`、`bits-and-data`、`hardware-software`、`troubleshoot-machine` 后解锁。
- 游戏中心第十一张卡，锚点 `virtual-computer-factory`。
- 全程是屏幕内虚拟部件，不要求触摸真实插头、电源、设备内部或拆机；出现电源和内部故障一律交给大人。

## 六个工位

1. **信息流水线**：把键盘输入、CPU 处理、屏幕输出按正确顺序连接。
2. **工作记忆站**：将正在编辑的作品放入内存，保存后复制到长期存储，重启时区分两者。
3. **比特灯板站**：根据位值点亮 0/1 组合，组成一个目标数字并识别 8 比特为一字节。
4. **软硬件搭档站**：为画图任务选择鼠标、屏幕等硬件和画图程序等软件，观察操作系统协调路线。
5. **安全诊断站**：从“静音、亮度、连接状态”等简单可逆原因开始，一次改变一项并复测。
6. **整机数据旅行**：把输入、CPU、内存、存储、输出和软件串成完整证据链，解释一次保存与显示。

## 玩法

- 每一步显示可视部件卡和数据流状态，孩子选择下一部件或安全动作。
- 正确操作进入“工位检测单”，错误操作保留机器状态并指出应观察的职责或数据流。
- 双击、未知动作和非当前阶段输入不会推进；六工位结束可轮换重玩。
- 无倒计时、生命值或排行榜。
