# 比特岛大冒险

「比特岛大冒险」是面向小学二年级儿童的电脑端互动启蒙课程。孩子跟随机器人向导比比探索九座岛屿，在短时、可操作的冒险中交替学习电脑基础、硬件原理、网络知识、数字创作、AI 素养和编程思维。

## 当前课程范围

- 展示九座岛屿、45 节课程的完整学习路线、目标与完成状态。
- 提供课程地图、本机进度、徽章、声音与动画设置，以及受保护的家长区。
- **九座岛屿的 45 节课程均可完整游玩**：覆盖电脑操作、文件管理、编程思维、事件变量函数、数字安全、硬件原理、数据表示、互联网、数字创作、在线协作、AI 素养与综合实践。
- 每节课采用约 8–10 分钟的故事、讲解、引导练习、独立挑战和奖励节奏；完成课程后可随时从地图重玩。
- 所有课程都通过统一的课程注册表接入地图、续课进度、完成徽章和通用完成页；只有已经实现并通过测试的课程才会标记为可玩。
- 课程地图还提供五次滚动学习计划、知识图鉴、45 词儿童电脑词典、45 道逐课概念配对题、18 道九岛情境题与孩子自主加练清单。
- 每课结束可选择“我会讲了 / 我想再练 / 请大人帮忙”，选择只保存在本机，并在家长区形成可直接打开的复习建议。
- GitHub Pages 版本支持安装和离线重访；首次联网打开后会预缓存当前页面、脚本、样式、清单和应用图标。

## 隐私

网站无需登录，不收集儿童的姓名、照片、联系方式或其他个人信息。学习进度、徽章、三选一自评与偏好设置只保存在当前浏览器的本机存储中，不会上传到服务器；家长可以随时在家长区重置、导出或恢复这些数据。备份导出采用字段白名单，不包含自由文本、姓名、账号或答题内容。GitHub Pages 版本使用内容安全策略禁止连接外部服务和加载第三方脚本。若浏览器无法使用本机存储，课程仍可继续，但本次进度不会持久保存。

## 本地运行

需要 Node.js `>=22.13.0`。从仓库检出后按锁文件安装依赖：

```bash
npm ci
```

常用命令：

```bash
npm run dev
npm test
npm run lint
npm run typecheck
npm run audit:prod
npm audit
npm run test:pages
```

- `npm run dev`：启动本地开发服务器。
- `npm test`：执行生产构建并验证渲染结果。
- `npm run lint`：检查代码质量与格式规则。
- `npm run typecheck`：严格检查课程、进度与发布代码的 TypeScript 类型。
- `npm run audit:prod`：阻止带有中高危运行时依赖的版本进入发布流程。
- `npm audit`：同时检查本地构建工具与运行时依赖。
- `npm run test:pages`：构建并验证部署在 `/computer-study/` 下的 GitHub Pages 静态版本。
