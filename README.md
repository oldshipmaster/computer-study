# 比特岛大冒险

「比特岛大冒险」是面向小学二年级儿童的电脑端互动启蒙课程。孩子跟随机器人向导比比探索四座岛屿，在短时、可操作的冒险中交替学习电脑基础和编程思维。

## 首版范围

- 展示四座岛屿、20 节课程的学习路线、目标与开放状态。
- 提供课程地图、本机进度、徽章、声音与动画设置，以及受保护的家长区。
- **首版只有第 1 课《键盘驾驶飞船》可以完整游玩。**其余 19 节课程会展示学习目标，并明确标记为「即将开放」。
- 第一课覆盖键盘方向键操作、顺序指令编排、即时引导和完成徽章。

## 隐私

网站无需登录，不收集儿童的姓名、照片、联系方式或其他个人信息。学习进度、徽章与偏好设置只保存在当前浏览器的本机存储中，不会上传到服务器；家长可以随时在家长区重置这些数据。若浏览器无法使用本机存储，课程仍可继续，但本次进度不会持久保存。

## 本地运行

需要 Node.js `>=22.13.0`。首次运行先安装依赖：

```bash
npm install
```

常用命令：

```bash
npm run dev
npm test
npm run lint
```

- `npm run dev`：启动本地开发服务器。
- `npm test`：执行生产构建并验证渲染结果。
- `npm run lint`：检查代码质量与格式规则。
